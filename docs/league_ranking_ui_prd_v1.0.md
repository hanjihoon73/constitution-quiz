# [PRD] 랭킹 페이지 타이틀 UI 개선 (v1.0)

## 1. 개요
현재 랭킹 페이지의 상단 타이틀 및 컨트롤 영역의 시인성과 미적 완성도를 높이기 위해 제공된 디자인 시안에 맞춰 UI를 개선합니다.

## 2. 목표
- 랭킹 페이지 상단에 서비스의 핵심 정보(리그 기간, 타이틀)를 직관적으로 배치
- 세련된 다크 모드 스타일의 튜브(캡슐) 디자인 적용
- 기능적 아이콘(도움말) 추가를 통한 사용자 경험 개선

## 3. 주요 요구사항
### 3.1 타이틀 영역 개선
- **서브타이틀 추가**: "주간 리그 랭킹" 메인 타이틀 아래에 "Weekly Leaderboard" 영문 텍스트 추가 (색상: #A1A1A1 추정, 폰트 크기: 11px)
- **리그 기간 스타일링**: 
    - 위치: 타이틀 영역 우측 하단 정렬
    - 형태: 캡슐(튜브) 모양 (Rounded-full)
    - 배경색: `#2D2D2D`
    - 글자색: `#FF8400`
    - 폰트 크기: 11px~13px (시안 기준)

### 3.2 컨트롤 영역 개선
- **도움말 아이콘 추가**: "새로고침" 버튼 좌측에 `lucide.circle-question-mark` 아이콘 추가
- **스타일 통일**: 기존 "나가기(ArrowLeft)", "새로고침(RefreshCw)" 버튼과 동일한 크기 및 색상 스타일 적용

## 4. 상세 UI 명세
- **리그 기간 포맷**: "YY.MM.DD ~ YY.MM.DD" (예: 26.03.16 ~ 26.03.23)
- **아이콘**: Lucide-react 라이브러리의 `CircleHelp` 또는 `CircleQuestion` (이미지에 따라 `circle-question-mark` 사용)

## 5. 기대 효과
- 사용자가 현재 리그 기간을 더 명확하게 인지할 수 있음
- 전반적인 앱의 디자인 완성도 향상 및 프리미엄 느낌 강조
