# [PRD] 랭커 카드 상세 통계 툴팁(Tooltip) (v1.2)

## 1. 개요
랭커 카드에서 생략된 상세 통계(퀴즈팩 완료 수, 정답률 등)를 사용자가 마우스 호버 또는 터치 시 풍선 도움말(Tooltip) 형태로 제공하여, 메인 UI의 깔끔함을 유지하면서도 필요한 정보는 전달할 수 있도록 합니다.

## 2. 목표
- 랭커 카드의 디자인 간결화 유지
- 필요 시에만 상세 정보를 노출하여 정보 과부하 방지
- 직관적인 아이콘과 배지 시스템을 통한 빠른 정보 인지

## 3. 주요 요구사항
### 3.1 툴팁 노출 조건
- **데스크톱**: 랭커 카드 영역에 마우스 호버(`mouseenter`) 시 노출, 마우스 이탈(`mouseleave`) 시 제거
- **모바일**: 랭킹 카드 터치 시 노출/제거 (토글 방식 제안)

### 3.2 툴팁 구성 요소 (좌측부터)
1. **유니크 팩 수**: `Box` 아이콘 + 우측 상단 다크 배지(검정 배경, 흰색 숫자)
2. **총 완료 횟수**: `CircleCheckBig` 아이콘 + 우측 상단 다크 배지(검정 배경, 흰색 숫자)
3. **평균 정답률**: 텍스트 형태 (예: "76.5%", 회색 계열)

### 3.3 디자인 사양
- **배경**: 흰색 (#FFFFFF)
- **둥글기**: `Rounded-2xl` 또는 시안에 맞춘 곡률
- **그림자**: 하단부 부드러운 그림자 효과 (`shadow-lg`)
- **말풍선 꼬리**: 하단 중앙에 작은 삼각형(Tail) 배치
- **배지 스타일**: 배경 `#2D2D2D`, 글자 흰색, 초소형 사이즈

## 4. 상세 UI 명세
- **위치**: 랭커 카드 상단 중앙에 고정 (카드 위로 떠 있는 형태)
- **애니메이션**: 페이드 인(`fade-in`) 및 살짝 올라오는 효과 (`slide-in-from-bottom`) 적용 가능

## 5. 기대 효과
- 핵심 데이터(XP)와 상세 데이터(퀴즈 지표)의 분리를 통한 고도화된 UI 제공
- 사용자 간의 경쟁 심리를 자극하는 구체적인 지표 노출
