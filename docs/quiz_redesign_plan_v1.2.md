# 모두의 헌법 퀴즈 - 디자인 개편 구현 계획서
> **버전**: v1.2
> **작성일**: 2026-03-02
> **관련 디자인**: 첨부된 '선다형 퀴즈' 이미지 기반

## 1. 개요
브라우저 툴의 오류로 인해 잘못된 시안(Blue Theme)으로 분석되었던 내용을 바로잡고, 사용자가 첨부한 정확한 기획 이미지를 바탕으로 구현 계획을 전면 수정합니다.
메인 컬러 포인트는 다크 그레이(`#2D2D2D`), 오렌지(`#FF8400`), 정답 색상인 시안(Cyan, `#38D2E3`), 오답 색상인 핑크(Pink, `#FB84C5`)입니다.

---

## 2. 레이아웃 및 내비게이션 형태 복구

### 1) 상단 헤더 (`QuizLayout.tsx`)
- 우측 X 버튼이 아니라 **좌측 화살표(`←`) 및 여백 라인**만 있는 심플한 구조입니다.

### 2) 상단 진행 번호 (`QuizNavigation.tsx`)
- 가로형 진행 바가 아니라 **원형 번호 칩 나열 방식**입니다.
- **기본 상태**: 투명 배경, 연한 회색 테두리(`#E5E7EB`), 회색 번호 텍스트.
- **현재 퀴즈 - 정오답 결과 전 (상태 A)**: 다크 그레이(`#2D2D2D`) 배경, 오렌지(`#FF8400`) 번호 텍스트.
- **현재 퀴즈 - 정답 결과 (상태 B)**: 다크 그레이(`#2D2D2D`) 배경, 시안(`#38D2E3`) 번호 텍스트.
- **현재 퀴즈 - 오답 결과 (상태 C)**: 다크 그레이(`#2D2D2D`) 배경, 핑크(`#FB84C5`) 번호 텍스트.
- **현재 퀴즈 아님 - 정답 결과 (상태 D)**: 시안(`#38D2E3`) 배경, 다크 그레이(`#2D2D2D`) 번호 텍스트.
- **현재 퀴즈 아님 - 오답 결과 (상태 E)**: 핑크(`#FB84C5`) 배경, 다크 그레이(`#2D2D2D`) 번호 텍스트.

## 6. 빈칸채우기 퀴즈 (`ChoiceBlank.tsx`) 디자인 상세

### 1) 구조 및 레이아웃
- 질문 텍스트 아래에 회색 바탕(`#F3F4F6`)의 지문 영역이 있으며, 지문 안의 빈칸 위치에 **모서리가 둥근 인라인 입력창 형태의 빈칸 박스 (`borderRadius: 9999px`)**가 삽입됩니다.
- 지문 영역 아래에 **선택 가능한 보기 버튼들**이 좌측 정렬로 나열됩니다 (마찬가지로 모서리가 둥근 캡슐 형태).

### 2) 상태별 색상 규칙 (빈칸 박스 및 하단 보기 버튼 공통)

| 상태 | 배경 (`bg`) | 테두리 (`border`) | 텍스트 (`text`) |
|------|-----------|----------------|----------------|
| **기본 (답 입력 전/선택 해제)** | 흰색 | 연한 회색(`#D2D2D2` 등) | 회색(`#9CA3AF`) |
| **선택 완료 (답 입력 후)** | 흰색 | 오렌지(`#FF8400`) | 오렌지(`#FF8400`) |
| **정답 (결과 화면)** | 흰색 | 시안(`#38D2E3`) | 시안(`#38D2E3`) |
| **오답 (결과 화면)** | 흰색 | 핑크(`#FB84C5`) | 핑크(`#FB84C5`) |

> **동작 상세**
> - 하단의 보기 버튼(기본: 회색 테두리/텍스트)을 클릭하면 해당 보기가 즉시 지문 속 빈칸으로 이동(채워짐)하는 시각적 효과를 줍니다.
> - **답 입력 후**: 선택한 보기 버튼과 지문 속 채워진 빈칸 박스 모두 **오렌지색 테두리와 텍스트**를 가집니다 (배경은 흰색 유지).
> - **채점 결과 화면**: 
>   - 정답인 경우: 선택했던 보기 버튼과 빈칸 박스가 **시안색(Cyan) 테두리와 텍스트**로 변경됩니다.
>   - 오답인 경우: 선택했던 보기 버튼과 빈칸 박스가 **핑크색(Pink) 테두리와 텍스트**로 변경됩니다.
> - **해설 및 정오답 텍스트 영역**: 다른 퀴즈 유형들과 동일하게 힌트/해설 컨테이너에 정답/오답 표시를 노출합니다.

## 4. 퀴즈 콘텐츠 및 기타 (`QuizContent.tsx`, `QuizActions.tsx`)

### 1) 퀴즈 타이틀 및 지문 / 유형 뱃지
- **유형 뱃지**: 회색 배경 캡슐형 라벨 `유형`.
- **타이틀**: 기본 텍스트 색상(다크 그레이).
- **지문 박스**: 회색 백그라운드(`#F3F4F6`), 둥근 테두리 안에 지문 텍스트 배치. 좌측에 오렌지색 선(border-left)은 없습니다.

### 2) 힌트 및 정답 표시 영역 (`QuizContent.tsx`)
- 보기들과 메인 액션 버튼(정답확인) 사이에 위치.
- **좌측**: `💡 힌트` 텍스트 (초기 오렌지/회색 혼합, 형태 단순).
- **우측**: 결과 출력 시, 시안색 `O 정답입니다.` 또는 핑크색 `X 오답입니다.` 문구를 힌트 텍스트와 동일 선상 우측에 배치 (양끝 정렬 flex-between 활용).
- **해설 영역**: 결과 출력 시 힌트 버튼 아래에 렌더링되며, 하늘색/핑크색 배경에 해설 텍스트 노출.

### 3) 하단 메인 액션 버튼 (`QuizActions.tsx`)
- 기존에 분할했던 (이전 / 다음) 버튼 구조가 아님. **하단 1개의 꽉 찬 버튼** 구조 유지.
- **답 입력 전**: 회색(Light Grey, `#F3F4F6` 계열) 배경, 연회색 텍스트(활성화 불가 느낌), "정답 확인"
- **답 입력 후**: 다크 그레이(`#2D2D2D`) 배경, 오렌지(`#FF8400`) 텍스트, "정답 확인"
- **결과 확인 후 (다음 퀴즈)**: 오렌지(`#FF8400`) 배경, 흰색 텍스트, "다음 퀴즈"

## 5. OX(참/거짓) 퀴즈 (`TrueFalse.tsx`) 디자인 상세

### 1) 구조 및 레이아웃
- 보기 텍스트("맞다", "틀리다" 등)는 출력하지 않고 **거대한 O, X 아이콘만 중앙에 배치**합니다.
- 버튼 형태: 정사각형에 가까운 큰 박스 형태 (`borderRadius: 16px` 정도).

### 2) 상태별 색상 규칙
| 상태 | 배경 (`bg`) | 테두리 (`border`) | 아이콘 색상 (`icon`) |
|------|-----------|----------------|----------------|
| **기본 (Default)** | 흰색 | 연한 회색(`#D2D2D2` 등) | 회색(`#9CA3AF`) |
| **선택 (Selected - 채점 전)** | 연한 오렌지(`#FFEEDB`) | 오렌지(`#FF8400`) | 오렌지(`#FF8400`) |
| **정답 (Correct - 결과 화면)** | 연한 시안(`#DAF5FF`) | 시안(`#38D2E3`) | 시안(`#38D2E3`) |
| **오답 (내가 선택한 틀린 답)** | 연한 핑크(`#FEE6F3`) | 핑크(`#FB84C5`) | 핑크(`#FB84C5`) |

> **동작 상세 (결과 화면)**
> - **정답을 맞춘 경우**: 선택한 정답 버튼은 시안(Cyan)으로 강조되고, 나머지 오답 버튼은 기본(Default) 상태 유지.
> - **오답을 고른 경우**: 내가 선택한 오답 버튼은 핑크(Pink)로 강조되고, 원래 정답이었던 버튼은 시안(Cyan)으로 강조됨.

## 7. 퀴즈팩 결과 화면 (`QuizPackResult.tsx`) 디자인 상세

### 1) 상단 레이아웃
- **뒤로가기 버튼 (`←`)**: 퀴즈 진행 화면의 나가기 버튼과 동일한 호버 효과(그림자 없이 위로 뜨는 효과). 클릭 시 홈 화면으로 이동.
- **축하 아이콘**: `party-popper` (Lucide) 아이콘 배치.
- **타이틀/서브타이틀**: "퀴즈팩 003 완료!", "수고하셨습니다."

### 2) 중앙 점수 레이아웃
- **원형 차트**: 정답률(50% 등)을 보여주는 도넛형 차트.
- **결과보기 버튼**: 우측 하단 배치. 아이콘 `search-check` (Lucide). 호버 시 그림자 없이 위로 뜨는 효과 적용.

### 3) 통계 및 평가 레이아웃
- **요약 카드**: 좌측부터 전체 퀴즈 수(블랙), 정답 수(시안색 `#38D2E3`), 오답 수(핑크색 `#FB84C5`) 표기.
- **평가 카드 (별점)**: "이번 퀴즈팩은 어땠나요?" 문구와 5개 별점 표시.

### 4) 하단 액션 버튼
- 모든 하단 메인 버튼은 호버 시 **위로 뜨는 효과와 그림자 효과**를 동시 적용.
- **다음 퀴즈팩 시작**: 오렌지색 배경, 흰색 텍스트.
- **다시 풀기**: 연한 회색(Light Grey) 배경, 짙은 회색 텍스트.
- **홈으로 가기**: 다크 그레이(Dark Grey) 배경, 오렌지색 텍스트.
