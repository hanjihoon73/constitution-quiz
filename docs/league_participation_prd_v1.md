# 리그 참여 기준 변경 PRD (v1)

## 1. 개요
* **목표**: 주간 리그에 랭크업되고, 참가 팝업이 뜨는 기준을 기존 '퀴즈팩을 완료했을 때'에서 '퀴즈팩을 완료하고 XP를 1 이상 획득했을 때' 로 변경.
* **필요성**: 기존 로직은 XP를 얻지 못하거나 이번 주에 단 한 번도 활동하지 않은 휴면유저(0 XP)까지 모두 랭킹 리스트에 노출되어 데이터 노이즈 및 사용자 경험 저하를 유발합니다. 활성 참여자(`weekly_xp > 0`)만 랭킹 대상으로 삼아 실제 경쟁의 질을 올립니다.

## 2. 변경 요구사항
1. **랭킹 대상 조건 변경**
   * 기존: 조건 없음 (`weekly_xp` = 0 인 전체 유저 노출)
   * 변경: `weekly_xp > 0` 조건 만족 시에만 결과 리스트에 노출.
2. **신규 참가 및 팝업 노출 조건 변경 로직(보존)**
   * 기존 로직 유지: 이번 주에 학습한 퀴즈팩 개수를 기준으로 참가 화면을 띄움으로써, 이미 완료한 퀴즈팩 재풀이로 XP 0점을 획득했더라도 "주간 리그에 다시 도전해 보세요"라는 취지의 참가 확인 노출은 의도대로 동작하게 만듭니다.
