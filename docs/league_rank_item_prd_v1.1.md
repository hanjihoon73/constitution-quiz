# [PRD] 랭커 카드(RankItem) UI 개편 (v1.1)

## 1. 개요
랭킹 페이지의 핵심 요소인 랭커 카드를 더 현대적이고 깔끔한 디자인으로 개편하여 정보의 위계와 시각적 즐거움을 제공합니다.

## 2. 목표
- 1~3위 상위 랭커에 대한 시각적 보상(메달) 강화
- 정보의 배치 최적화를 통해 쾌적한 가독성 확보
- 불필요한 장식 요소를 제거하고 핵심 데이터(XP)에 집중

## 3. 주요 요구사항
### 3.1 순위 표시 (좌측)
- **1위**: `medal_gold.svg` 배경 + 숫자 '1' (흰색, 중앙 배치)
- **2위**: `medal_silver.svg` 배경 + 숫자 '2' (흰색, 중앙 배치)
- **3위**: `medal_bronze.svg` 배경 + 숫자 '3' (흰색, 중앙 배치)
- **4위 이하**: 숫자만 표시 (기존 스타일 유지하되 폰트 조정)

### 3.2 사용자 정보 표시 (중앙)
- **닉네임**:  Bold 스타일, 검은색 폰트
- **타이틀 아이콘**: 사용자의 칭호 아이콘 표시
- **타이틀(칭호)**: 다크 캡슐(배경: `#2D2D2D`, 글자: 흰색) 형태로 닉네임 우측에 배치

### 3.3 XP 정보 표시 (우측)
- **주간 XP**: 상단 배치, 주황색(`#FF8400`), Bold, 큰 폰트
- **누적 XP**: 하단 배치, 회색, 작은 폰트 (형식: "누적 XP 3,650" 또는 "누적 XP 3.65K")

### 3.4 제거 항목
- 기존의 박스 아이콘, 체크 아이콘 및 정답률 통계 제거 (디자인 간소화)

## 4. 상세 UI 명세
- **카드 스타일**: 배경 흰색, 옅은 회색 테두리, Rounded-xl
- **레이아웃**: `Flex`를 사용한 가로 배치 (순위 - 사용자 정보 - XP 정보)
- **여백**: 시안의 비율에 맞춰 적절한 패딩 및 간격 조정

## 5. 기대 효과
- 상위 랭커의 성취감 고취
- 모바일 환경에서의 가독성 극대화
