# 랭킹 페이지 자동 포커스 PRD (v1)

## 1. 개요 (Overview)
* **목표**: 사용자가 주간 리그 랭킹 페이지에 진입했을 때, 본인의 랭킹 항목이 화면의 중앙에 위치하도록 자동으로 부드럽게 스크롤해줍니다.
* **필요성**: 랭킹 데이터가 길어질 경우, 본인의 위치를 찾기 위해 직접 스크롤을 내려야 하는 불편함을 없애고 사용자 경험(UX)을 개선하기 위함입니다.

## 2. 세부 요구사항 (Requirements)
1. **대상 화면**: `src/app/league/page.tsx` (주간 리그 랭킹 페이지)
2. **동작 조건**:
   * 서버에서 랭킹 데이터를 로드 완료하여 목록이 화면에 렌더링된 시점.
   * 항목 중 자신의 랭킹을 나타내는 데이터(`isMe: true`)가 존재할 때.
3. **효과**:
   * 즉발적인 팝업 스크롤 대신 `smooth` 애니메이션 모드로 화면을 이동.
   * `block: 'center'` 옵션을 사용하여 사용자의 항목이 모바일 화면 세로축의 중앙에 위치하도록 구성.
   * 데이터 로드 후 애니메이션(약 300~400ms)이 이루어지는 것을 감안해 렌더링 직후 약간의 타이머를 두고 실행.

## 3. 예외 처리
* 나의 랭킹이 표시되지 않는 상태(로그인 에러, 혹은 랭킹 미참여) 조건에서는 동작하지 않습니다.

## 4. 파급 효과 (Impact)
* **컴포넌트 추가/변경**: 기존 `RankItem` 내부에만 State Effect를 부여하므로 레이아웃이나 기존 API 호출 로직 등에 영향을 주지 않습니다.
