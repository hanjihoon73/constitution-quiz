- **주간 리그 시스템**
    1. 기간: 이번 주 월요일 0시 ≤ 운영 기간 < 다음 주 월요일 0시
    2. 운영 방식
        1. 랭킹 처리:
            1. 1차: 리그 기간 중 사용자가 획득한 XP 비교 → `users.weekly_xp`
            2. 2차: XP가 같으면, 완료한 퀴즈팩 개수(unique) 비교 → `sum(users_quizpacks.completed_count >= 1)`
            3. 3차: 완료한 퀴즈팩 개수(unique)가 같으면, 퀴즈팩 완료 횟수의 합계 비교 → `sum(users_quizpacks.completed_count)`
            4. 4차: 퀴즈팩 완료 횟수의 합계가 같으면, 사용자의 퀴즈팩 평균 정답률 비교 → `users.quizpack_avrg_correct`
        2. 
        3. 리그 기간 중 사용자의 `users.weekly_xp`가 업데이트 될 때마다 랭킹 실시간 업데이트
        4. 월요일 0시에 모든 사용자의 랭킹 초기화
        5. 랭킹 초기화 후 로그인 또는 홈으로 이동하면, 지난주 리그 결과를 팝업으로 표시
        6. 주간 리그 시작 후 퀴즈팩을 완료하면 주간 리그에 자동 참여되고, 퀴즈팩 완료 후 홈으로 이동하면, 주간 리그 참여 안내 팝업 표시 (XP 획득 여부와 관계없음)
    3. DB 테이블:
        1. 사용자의 퀴즈팩 평균 정답률 관리 → `users.quizpack_avrg_correct`
        2. 주간 랭킹 관리 테이블 → `users_weekly_ranking` 
        3. 사용자별 랭킹 히스토리 관리 테이블 → `users_ranking_history` 
            1. mendatory columns: `weekly_xp` = `users.weekly_xp` 
            2. (중요) 매 주 월요일 0시에 `user_ranking_history.weekly_xp`에 `users.weekly_xp` 데이터 저장 후 `user.weekly_xp` 초기화
    
- **리그 관련 UI**
    1. 리그 플로팅 버튼 추가
        1. 위치: 홈페이지, 마이페이지
        2. 스타일: 원형 뱃지, 배경 #2D2D2D, 테두리 #FF8400, 아이콘(`lucide.trophy`), 아이콘 색상 #FF8400
    2. 주간 리그 종료 안내 팝업
        1. 레이아웃: 현재 사용 중인 모달
        2. 아이콘: `lucide.trophy`  (#2D2D2D)
        3. 타이틀: “지난 주 리그가 종료됐습니다!”
        4. 기간: yy.mm.dd 월 0시 ~ yy.mm.dd 월 0시
        5. 나의 랭킹: “n위”
        6. 순위에 따른 안내문구:
            1. 1위: "역사적인 순간! #{닉네임}님. 주간 리그 랭킹 1위에 등극했습니다."
            2. 2위: “가문의 영광! #{닉네임}님. 주간 리그 랭킹 2위에 등극했습니다."
            3. 3위: "대단합니다! #{닉네임}님. 주간 리그 랭킹 3위에 등극했습니다."
            4. 4위: "엄청나네요! #{닉네임}님. 주간 리그 랭킹 4위를 달성했습니다."
            5. 5위: “놀라워요! #{닉네임}님. 주간 리그 랭킹 5위를 달성했습니다."
            6. 5위 미만: “수고하셨습니다! #{닉네임}님. 주간 리그 랭킹 n위를 달성했습니다."
        7. 버튼: [주간 랭킹 확인] → 클릭 → 랭킹 페이지로 이동
        8. 화면 우상단에 닫기 버튼 [X]
        9. [주간 랭킹 확인] 또는 닫기 버튼 클릭 시 이후 재등장 없음 (쿠키 초기화 시 재등장)
        10. 주간 리그 시작 안내 팝업과 겹치는 경우, 주간 리그 종료 안내 팝업을 우선 표시 → 닫기 또는 [주간 랭킹 확인] → 홈으로 이동 → 주간 리그 시작 안내 팝업 표시
    3. 주간 리그 시작 안내 팝업
        1. 레이아웃: 현재 사용 중인 모달
        2. 아이콘: `lucide.trophy`  (#2D2D2D)
        3. 타이틀: “새로운 주간 리그가 시작됐습니다!”
        4. 기간: yy.mm.dd 월 0시 ~ yy.mm.dd 월 0시
        5. 안내문구: “퀴즈팩을 완료하고 XP를 획득해 상위 랭킹에 등극하세요.”
        6. 버튼: [주간 랭킹 확인] → 클릭 → 랭킹 페이지로 이동
        7. 화면 우상단에 닫기 버튼 [X]
        8. [주간 랭킹 확인] 또는 닫기 버튼 클릭 시 이후 재등장 없음 (쿠키 초기화 시 재등장)
    4. 랭킹 페이지
        1. 헤더: 홈/마이페이지와 동일 컴포넌트 사용
        2. 렌더링: 화면 페이드인 및 구성 요소가 순차적으로 조립되는 효과
        3. [나가기] 버튼 → 마이페이지와 같은 위치 및 스타일
        4. [새로고침] 버튼 → 터치 → 현재 사용자들의 주간 랭킹 업데이트 (`users_weekly_ranking`)
        5. 랭커 목록
            1. 개수: 20명
            2. 1위~5위 고정 노출 6위 미만은 조회 사용자의 랭킹을 기준으로 상위 7명, 하위 7명 표시
            3. 목록 표시 정보:
                1. 현재 랭킹
                2. 사용자 타이틀 뱃지
                3. 사용자 타이틀
                4. 닉네임
                5. 주간 XP
                6. 누적 XP
                7. 완료한 퀴즈팩 개수: 아이콘 `lucide.box`,  아이콘 우상단에 갯수 뱃지 표시
                8. 퀴즈팩 완료 횟수: 아이콘 `lucide.circle-check-big`, 아이콘 우상단에 갯수 뱃지 표시
                9. 퀴즈팩 평균 정답률: n% (소수점 1자리까지)
            4. 각 목록에 버튼 스타일 UI 적용 (호버 효과 포함, 클릭 이벤트 없음)
            5. 조회 사용자의 목록에 자동 포커싱 (목록 버튼 테두리 #FF8400)
